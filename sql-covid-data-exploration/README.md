# COVID-19 Data Exploration (SQL)

## Overview
This project explores global COVID-19 data using SQL to identify trends in cases, deaths, and vaccination progress across different countries and regions.

The goal of this project was to practice exploratory data analysis using SQL queries and analytical functions to better understand patterns in the dataset.

Dataset source:  
https://ourworldindata.org/covid-deaths

---

## Objectives
- Explore global COVID-19 trends
- Compare infection and death rates across countries
- Analyse vaccination progress
- Practice SQL analytical queries and aggregations

---

## Techniques Used
- Filtering and sorting data
- Aggregate functions (`SUM`, `MAX`, `COUNT`)
- Joins
- Window functions
- Common Table Expressions (CTEs)
- Temporary tables
- Creating SQL views for later analysis

---

## Example Analyses
The project explores several questions, including:

- Total COVID-19 cases and deaths by country
- Percentage of population infected
- Countries with the highest infection rates
- Global death percentages over time
- Vaccination progress by country
